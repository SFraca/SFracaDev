# SFracaDev - Portfolio Personal

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![License: Apache 2.0](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

Bienvenido al repositorio de SFracaDev, un portfolio web personal diseñado con una estética moderna basada en la terminal de comandos. Este proyecto destaca por su enfoque en la experiencia de usuario, un diseño responsivo impecable y una navegación intuitiva.

## Características Principales

- **Estética Terminal**: Interfaz única inspirada en terminales de comandos, con elementos interactivos y animaciones fluidas.
- **Diseño 100% Responsivo**: Optimizado para PC, Tablets (iPad) y Dispositivos Móviles, garantizando una visualización equilibrada en cualquier resolución.
- **Filtrado de Proyectos**: Galería de trabajos organizada con sistema de filtrado por tecnologías.
- **Sección de Habilidades**: Presentación visual de skills técnicos con categorización clara.
- **Páginas de Error Personalizadas**: Gestión completa de errores HTTP (403, 404, 500, etc.) con mensajes claros y coherentes con el diseño.
- **Formulario de Contacto**: Integración lista para el envío de mensajes.
- **Sección de Enlaces**: Hub centralizado de redes sociales y recursos externos.

## Tecnologías Utilizadas

- **Frontend**: 
  - HTML5 Semántico
  - CSS3 (Variables nativas, Flexbox, Grid, Media Queries)
  - JavaScript Vanilla (Lógica de filtrado, navegación y responsive)
- **Backend**: 
  - PHP (Procesamiento de formularios)

## Estructura del Proyecto

```bash
.
├── error/              # Páginas de error personalizadas (404, 500, etc.)
├── images/             # Activos visuales y capturas de proyectos
├── js/                 # Lógica de componentes y efectos
├── php/                # Scripts de backend para el formulario
├── projects/           # Páginas individuales de detalle de proyectos
├── resources/          # Descargables (CV, documentos)
├── index.html          # Página principal
├── projects.html       # Listado general de proyectos
├── main.css            # Estilos globales y diseño responsivo
└── README.md           # Documentación del proyecto
```

## Instalación Local

1. Clona este repositorio:
   ```bash
   git clone https://github.com/SFraca/SFracaDev.git
   ```
2. Abre el archivo index.html en tu navegador preferido o utiliza una extensión como "Live Server" en VS Code.

*Nota: Para que el formulario de contacto funcione correctamente, se requiere un entorno de servidor con soporte para PHP.*
